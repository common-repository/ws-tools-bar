== Description ==
This plugin adds a Quick toolbar on the top of blog. 
direct acces to Dashboard, Write, Edit, Awaiting Moderation...
it makes Administrators Operating too easy and quickly.

== Installation ==
1. Upload the full directory into your wp-content/plugins directory.
2. Activate the Plugin in WP-Admin. 
3. Enjoy!
